NapCamp - Hipcamp clone

NapCamp is a parody clone of the website Hipcamp. Users can explore different Bay Area locations where they can nap. User's can book beds, create, edit and delete reviews on each location, as well as view the location of spots through googlemaps.

Live link Github repo

AfterHours - Discord clone

After Hours is clone of the app Discord. The project was created over a two week period with two other members wherein we enabled live chat functionality through flack socket io as they join/ create servers and channels. Users can also search and direct chat other users.

Live link Github Repo

Outergram - Instagram clone

Outergram app is a clone of instagram.com. The app enables users to post images and connect with others through their posts. Users can like and comment on posts as well as follow, message and view other users posts.
